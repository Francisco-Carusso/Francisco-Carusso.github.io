Creamos las hojas de estilos Dentro del head enlazamos la hoja de estilo con la etiqueta <link>
  • <link rel="stylesheet" href="/style.css">
  • Atributo rel="stylesheet" tipo de documenta al que estoy llamando.
  • Atributo href="/style.css" (dentro de las comillas va la dirección del archivo
  • Este es la forma recomendable para los estilos.
  • Acontinuación escribimos el css en su respectivo archivo.

Tipos de tamaños: Unidades de medida
Pixeles Estructura del código HTML: <article class="contenedor">
<p class="pixeles">Unidad de medida en pixeles.</p> </article> Estructura del código CSS3: .pixeles {
/* 16px es el valor por defecto */
font-size: 16px;
background-color: gray;
width: 400px; }

Unidad de medida en pixeles.
Porcentaje Estructura del código HTML: <article class="contenedor">
<p class="porcentaje">Unidad de medida en porcentaje.</p> </article> Estructura del código CSS3: .porcentaje {
/* 100% del valor contenedor 16px */
font-size: 150%;
background-color: gray;
width: 50%; }

Unidad de medida en porcentaje
EM Estructura del código HTML: <article class="contenedor">
<p class="em">Unidad de medida "em".</p> </article> Estructura del código CSS3: .em {
/* 1em = valor del contenedor */
font-size: 1.5em;
background-color: gray;
width: 5em; }

Unidad de medida "em"
REM Estructura del código HTML: <article class="contenedor">
<p class="rem">Unidad de medida "rem".</p> </article> Estructura del código CSS3: .rem {
/* 1rem = valor del root (valor por defecto que son 16px) */
font-size: 0.8rem;
background-color: gray;
width: 5rem; }

Unidad de medida "rem"
